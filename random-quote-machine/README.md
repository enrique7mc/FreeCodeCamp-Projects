# Random Quote Machine

Show a famous quote at random order using data from
[Quotes API](https://andruxnet-random-famous-quotes.p.mashape.com/).

## Features
- Normalize.css
- jQuery to perform Ajax calls
- Event handlers using vanilla JavaScript
- CSS transitions
